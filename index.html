Linguagem de Marcação de Hipertexto

O que é HTML?

HTML é uma linguagem de marcação, não uma linguagem de programação. Isso significa que ela não executa funções lógicas, mas sim define a estrutura e o significado do conteúdo em uma página web.

Estrutura básica de um documento HTML:

  1. Documento:

    • Um documento HTML começa com a declaração do tipo de documento () e é envolvido pela tag .
    • Cabeçalho ( ): Contém informações sobre a página, como título, metadados, links para folhas de estilo (CSS) e scripts (JavaScript). Não é exibido diretamente no conteúdo principal da página.
    • Corpo (): Contém todo o conteúdo visível da página, como textos, imagens, links, etc.

    Elementos e tags:

    • Elementos HTML são definidos por tags, que são palavras-chave envolvidas por colchetes angulares < >, como < p > para parágrafo, < h1 > para título principal, < img > para imagem, < a > para link, etc.
    • Tags podem ter atributos que fornecem informações adicionais, como o atributo src na tag < img > que especifica o caminho da imagem, ou o atributo href na tag < a > que define o destino do link.
    • Um elemento HTML consiste na tag de abertura, o conteúdo e a tag de fechamento (com exceção de algumas tags como < img > e < br >).

    Importância do HTML:

    • É a base da web e essencial para a criação e estruturação de páginas web
    • Permite que os desenvolvedores organizem o conteúdo de forma lógica e visual, criando uma experiência rica para os usuários.
    • É fácil de aprender e entender, com muitos recursos e documentação disponíveis.

    HTML5:

    • A versão mais recente do HTML, que adicionou novas tags semânticas (como < article >, < header >, < footer >) e novas funcionalidades, como suporte a vídeo e áudio.

    A Importância da Acessibilidade no HTML:

    • A Importância da Acessibilidade no HTML A acessibilidade no HTML é fundamental para garantir que todas as pessoas, independentemente de suas habilidades ou deficiências, possam acessar, entender e interagir com o conteúdo da web.
    • A Importância da Acessibilidade no HTML A acessibilidade no HTML é fundamental para garantir que todas as pessoas, independentemente de suas habilidades ou deficiências, possam acessar, entender e interagir com o conteúdo da web. Isso significa projetar e desenvolver sites de forma que sejam utilizáveis por indivíduos com deficiências visuais, auditivas, motoras ou cognitivas. Ignorar a acessibilidade é excluir uma parcela significativa da população e limitar o alcance do seu site.
